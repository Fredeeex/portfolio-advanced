# 💼 Dan Fred — Premium Portfolio (Glass UI)

A premium, client-ready portfolio website built using **HTML, CSS, and Vanilla JavaScript**.

Includes **Dark/Light toggle**, **modern glassmorphism UI**, **strong animations**, **project modal popup**, and **live preview buttons**.

---

## 🌍 Live Demo

🔗 **Demo:**  
https://fredeeex.github.io/portfolio-advanced/

---

## ✨ Features

- Responsive (Mobile-First)
- Sticky navbar + mobile menu
- Dark/Light theme toggle (saved in LocalStorage)
- Glassmorphism UI (modern premium look)
- Smooth scroll + scroll progress indicator
- Strong reveal animations (staggered)
- Animated skills progress bars (on view)
- Project filtering (tabs)
- Project modal popup (details)
- Live demo + repo buttons per project

---

## 🛠 Tech Stack

- HTML5
- CSS3 (Flexbox + Grid)
- Vanilla JavaScript (ES6)
- Intersection Observer API
- LocalStorage (theme persistence)
